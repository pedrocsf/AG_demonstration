Legenda - Conceitos Genéticos

  • Indivíduo: Uma única solução possível para o problema (um ponto na tela).
  • População: Conjunto de todos os indivíduos avaliados na geração atual.
  • Cromossomo / Genótipo: A estrutura de dados do indivíduo (aqui, as coordenadas X e Y).
  • Gene: Uma parte do cromossomo (ex: apenas a coordenada X).
  • Fenótipo: A manifestação do genótipo no ambiente (a posição visual no espaço 2D).
  • Melhor Solução: O indivíduo com o melhor "Fitness" (menor distância para o alvo).
